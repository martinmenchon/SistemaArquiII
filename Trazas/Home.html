REFERENCIAS 

					

					

						
El Código aceptado es una adaptación del lenguaje C++.
Se reconoce la sintaxis de la siguiente manera:

-  Palabras reservadas en minúsculas (for, while).
-  Variables  se deben escribir en mayúsculas (A[], X, Y, B[]).
-  El iterador se deben escribir en minúsculas (i, j, etc).
-  El iterador del while inicia en 1.
-  El incremento del while debe ir al final de la iteración (i++).
-  Se aceptan comentarios de linea empezando con //

Ejemplos de código reconocido:

for (i=1; i<= 5 ; i++) {		while (i<= 5){
	A[i] = E[i-1] + 6;			A[i] = E[i-1] + 6;
	B[i] = A[i] * Z; 			B[i] = A[i] * Z;
	C[i] = B[i-1] + X; 			C[i] = B[i-1] + X;
	D[i] = C[i] + Y; 			D[i] = C[i] + Y;
	E[i] = B[i] * D[i]; 			E[i] = B[i] * D[i];
} 						i++;
						}
